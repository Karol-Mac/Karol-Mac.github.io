Nutrinexus

brief overview

The application is a major project (specifically an engineering paper), carried out in a group of 3 people. The purpose of the application is to compare prices and nutritional values of food products.
Our aim was to make a platform where sellers could easily upload up-to-date product data and users could use a simple tool to find the best offers.
In addition to searching, comparing and adding products to favourites, users can also comment on the app and report defective products or rude comments.
The smooth and transparent operation of the application is supervised by administrators, for whom a special user management panel has been prepared. They can block inappropriate comments, block or even delete a user or defective product.

Main functionalities

  • Available for Everyone:
    • Browse and search for products
    • Compare product prices and nutritional values
    • View product details
    • See comments
    • Register new private/company Account
  • User Actions:
    • Log in and manage their profile
    • Add products to favourites
    • Manage the favourite list
    • Comment on application
    • Report inappropriate comments or defective products
  • Seller Actions:
    • Manage product data (CRUD)
    • See statistics about added products
  • Admin Actions:
    • Block or delete users
    • Remove inappropriate comments or products
    • Manage user accounts
    • Change the role of selected user
See usage of application in Postman

StyleSwap

brief overview

StyleSwap is a complex level 4 (on the Fielding scale) REST API for a shopping platform to sell clothes to other users. It is my solo project, which I am successively developing, fixing bugs and adding more features.
The API is as developer-friendly as possible, thanks to its simple design, the use of HATEOAS (which makes it much easier to add a frontend) and clear documentation created in the widely used SwaggerUI.
Users have access to a wide range of functionalities, including converting with sellers (other users), saving interesting clothes in their warehouse and performing real transactions (thanks to the connection with Stripe, the payment gateway).

Main functionalities

  • Available for Everyone:
    • Browse and search for clothes
    • View clothe details
    • Create an account
  • User Actions:
    • Log in and manage their profile
    • add and manage own clothing
    • Message to other sellers
    • Manage conversations
    • add clothes to the storage
    • buy the chosen clothing
  • Admin Actions:
    • Manage user accounts
    • Create new clothe categories
    • Manage orders (statistics)
